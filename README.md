#EPIA

## 简介

[EPIA](http://192.168.12.150:8888/epia) 是一个基于xml文件的接口解析框架，更准确的说它是一款持久层框架。EPIA以XML文件作为接口的基本单位，可灵活定义参数和数据操作行为，并可以在接口变更后做到即时生效。
